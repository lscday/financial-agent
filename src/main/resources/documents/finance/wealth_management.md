# 理财产品文档 - 金融资管产品说明书

## 一、产品概况

### 1.1 现金管理类产品
- **产品名称**: 天天利现金管理产品
- **产品类型**: 货币市场类
- **风险等级**: R1（低风险）
- **预期收益率**: 2.5%-3.0%
- **起投金额**: 1元
- **赎回方式**: T+0实时赎回，单日限额100万元
- **适合投资者**: 风险承受能力较低的投资者，适合流动性资金管理

### 1.2 固定收益类产品
- **产品名称**: 稳健增长理财产品
- **产品类型**: 固定收益类
- **风险等级**: R2（较低风险）
- **预期收益率**: 3.8%-4.2%
- **投资期限**: 90-365天
- **起投金额**: 10,000元
- **适合投资者**: 追求稳健收益、风险偏好较低的投资者

### 1.3 混合类产品
- **产品名称**: 灵活配置混合型理财产品
- **产品类型**: 混合类
- **风险等级**: R3（中等风险）
- **预期收益率**: 4.5%-6.0%
- **投资期限**: 180-730天
- **起投金额**: 50,000元
- **适合投资者**: 具有一定风险承受能力、追求较高收益的投资者

### 1.4 权益类产品
- **产品名称**: 价值成长股票型理财产品
- **产品类型**: 权益类
- **风险等级**: R4（较高风险）
- **预期收益率**: 8.0%-15.0%
- **投资期限**: 建议持有2年以上
- **起投金额**: 100,000元
- **适合投资者**: 风险承受能力较强、追求长期资本增值的投资者

## 二、风险揭示
1. 理财产品非银行存款，存在投资风险
2. 历史业绩不代表未来表现
3. 投资者应充分认识风险，谨慎投资
4. 产品净值可能受市场波动影响

## 三、费用说明
- 管理费：0.3%-0.8%/年
- 托管费：0.05%-0.1%/年
- 申购费：0%-1.0%（视产品类型）
- 赎回费：0%-0.5%（持有期限越长费率越低）
